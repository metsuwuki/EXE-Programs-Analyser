# Download-ready package

This folder is ready for GitHub upload so users can download and run without Rust.

## Start
- Double-click `Start Analyzer GUI.cmd`
- or run `exe_tester_gui.exe`

## Included
- `exe_tester_gui.exe` (main GUI)
- `exe_tester.exe` (scan engine)
- `logo.png`, `EXE_icon.ico`
- `logs/` (output reports)
